Branching

Branching is the method used in Git to have multiple edits and editors at the same time on a project. Branches can be split off the master, and reconciled with the master and other branches further down the line. A branch is a duplicate of the point on the master (or other branch) from which it split off.

Using branching

To see branches in your git, type this command:

  • git branch

To create one, use this command:

  • git branch (insert branch name)
