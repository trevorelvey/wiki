Box Model

Box model refers to the design and layout of HTML. It wraps around an element in HTML and provides a margin, border, padding, then the element content, be it a picture or text.

Parts of a box model

  • margin: it is outermost part if the box model. It is transparent.
  • border: inside in the margin it surrounds the padding and content that can have color applied to it.
  • padding: provides added space around the content between it and the border. Like the margin, it is also transparent.
  • content: the innermost element where the picture, text, etc appears.
